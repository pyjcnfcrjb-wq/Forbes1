This glossary collects the most important terms you'll encounter across the knowledge base. Whether you're brushing up on a concept before diving into an article or looking for a quick refresher mid-read, you can use the sections below to jump to the category that's most relevant. Each definition is written to be clear and accessible — no prior expertise required.

## 💻 Technology

<AccordionGroup>
  <Accordion title="Algorithm">
    An algorithm is a step-by-step set of instructions that a computer follows to solve a problem or complete a task. Algorithms are the foundation of all software — from sorting a list of names to recommending videos on a streaming platform.
  </Accordion>

  <Accordion title="API (Application Programming Interface)">
    An API is a set of rules and protocols that allows different software applications to communicate with each other. When you use a weather app that pulls data from a meteorological service, it's doing so through an API.
  </Accordion>

  <Accordion title="Bandwidth">
    Bandwidth refers to the maximum rate at which data can be transmitted over a network connection, typically measured in megabits per second (Mbps) or gigabits per second (Gbps). Higher bandwidth means more data can flow through the connection at once, which generally results in faster internet speeds.
  </Accordion>

  <Accordion title="Binary">
    Binary is a base-2 number system that uses only two digits — 0 and 1 — to represent all data in computers. Every piece of text, image, video, or instruction a computer processes is ultimately encoded as a sequence of these binary digits, called bits.
  </Accordion>

  <Accordion title="Cloud Computing">
    Cloud computing refers to the delivery of computing services — including storage, processing power, and software — over the internet rather than on local hardware. Instead of running applications on your own device, you access them through remote servers managed by providers like AWS, Google Cloud, or Microsoft Azure.
  </Accordion>

  <Accordion title="DNS (Domain Name System)">
    DNS is the internet's address book, translating human-readable domain names (like `example.com`) into the numerical IP addresses that computers use to identify each other. Without DNS, you'd need to memorize a string of numbers to visit every website.
  </Accordion>

  <Accordion title="Encryption">
    Encryption is the process of converting readable data (plaintext) into an unreadable format (ciphertext) using a cryptographic algorithm, so that only authorized parties with the correct key can decode it. It's the technology that keeps your passwords, messages, and financial transactions secure online.
  </Accordion>

  <Accordion title="Firewall">
    A firewall is a security system — either hardware, software, or both — that monitors and controls incoming and outgoing network traffic based on predefined security rules. It acts as a barrier between a trusted internal network and untrusted external networks, such as the internet.
  </Accordion>

  <Accordion title="HTTP / HTTPS">
    HTTP (HyperText Transfer Protocol) is the foundational protocol used for transmitting web pages over the internet. HTTPS is the secure version, which encrypts data in transit using TLS/SSL — you can identify it by the padlock icon in your browser's address bar.
  </Accordion>

  <Accordion title="IP Address">
    An IP (Internet Protocol) address is a unique numerical label assigned to every device connected to a network. It serves two main functions: identifying the host or network interface, and providing the location of that device in the network for routing purposes.
  </Accordion>

  <Accordion title="Machine Learning">
    Machine learning is a branch of artificial intelligence in which systems learn from data to improve their performance on tasks without being explicitly programmed for each scenario. It powers applications like spam filters, product recommendations, image recognition, and language translation.
  </Accordion>

  <Accordion title="Malware">
    Malware (short for "malicious software") is any software intentionally designed to disrupt, damage, or gain unauthorized access to a computer system. Common types include viruses, ransomware, spyware, and trojans.
  </Accordion>

  <Accordion title="Open Source">
    Open source refers to software whose source code is made freely available for anyone to view, use, modify, and distribute. Projects like Linux, Python, and Firefox are open source, meaning they benefit from contributions by developers worldwide.
  </Accordion>

  <Accordion title="Protocol">
    In networking, a protocol is a standardized set of rules that determines how data is transmitted and received between devices. Common protocols include TCP/IP (for general internet traffic), HTTP (for web pages), and SMTP (for email).
  </Accordion>

  <Accordion title="VPN (Virtual Private Network)">
    A VPN creates an encrypted tunnel between your device and a remote server, masking your IP address and protecting your data from interception. It is commonly used to enhance privacy online, access region-restricted content, and secure connections on public Wi-Fi.
  </Accordion>
</AccordionGroup>

## 🔬 Science

<AccordionGroup>
  <Accordion title="Atom">
    An atom is the smallest unit of a chemical element that retains the properties of that element. Every physical substance is made of atoms, which consist of a nucleus (containing protons and neutrons) surrounded by a cloud of electrons.
  </Accordion>

  <Accordion title="Biodiversity">
    Biodiversity refers to the variety of life found in a particular habitat or on Earth as a whole, including the diversity of species, genes, and ecosystems. High biodiversity is associated with more resilient and productive ecosystems.
  </Accordion>

  <Accordion title="Cell">
    A cell is the basic structural and functional unit of all living organisms. Some organisms, like bacteria, consist of a single cell, while humans contain trillions of cells that work together in tissues and organs.
  </Accordion>

  <Accordion title="DNA (Deoxyribonucleic Acid)">
    DNA is the molecule that carries the genetic instructions for the development, functioning, growth, and reproduction of all known living organisms. It is structured as a double helix and is found in the nucleus of cells.
  </Accordion>

  <Accordion title="Ecosystem">
    An ecosystem is a community of living organisms (plants, animals, microbes) interacting with each other and their physical environment (soil, water, air). Ecosystems range in scale from a small pond to an entire tropical rainforest.
  </Accordion>

  <Accordion title="Evolution">
    Evolution is the process of change in the heritable characteristics of biological populations over successive generations. Driven primarily by natural selection, it explains the diversity of life on Earth and the common ancestry of all organisms.
  </Accordion>

  <Accordion title="Gene">
    A gene is a segment of DNA that encodes the instructions for producing a specific protein or functional RNA molecule. Genes are the basic units of heredity, passed from parents to offspring and influencing traits like eye color, height, and disease susceptibility.
  </Accordion>

  <Accordion title="Gravity">
    Gravity is the fundamental force of attraction that acts between all objects with mass. On Earth, gravity pulls objects toward the planet's center, giving them weight; on a cosmic scale, it governs the orbits of planets, stars, and galaxies.
  </Accordion>

  <Accordion title="Molecule">
    A molecule is a group of two or more atoms held together by chemical bonds. Molecules can be simple (like water, H₂O) or extremely complex (like proteins containing thousands of atoms), and they are the building blocks of all chemical compounds.
  </Accordion>

  <Accordion title="Natural Selection">
    Natural selection is the mechanism by which individuals with traits better suited to their environment tend to survive and reproduce more successfully than others, passing those advantageous traits to future generations. It is the primary driver of evolutionary change, as described by Charles Darwin.
  </Accordion>

  <Accordion title="Photosynthesis">
    Photosynthesis is the process by which green plants, algae, and some bacteria convert sunlight, water, and carbon dioxide into glucose and oxygen. It is the foundation of most food chains on Earth and a critical component of the global carbon cycle.
  </Accordion>
</AccordionGroup>

## 🌍 Society

<AccordionGroup>
  <Accordion title="GDP (Gross Domestic Product)">
    GDP is the total monetary value of all goods and services produced within a country's borders in a specific time period. It is the most widely used measure of an economy's size and health, though it does not capture income inequality or quality of life.
  </Accordion>

  <Accordion title="Globalization">
    Globalization is the process by which businesses, cultures, and governments become increasingly interconnected across national borders through trade, communication, migration, and technology. It has accelerated dramatically since the late 20th century, driven by advances in transportation and the internet.
  </Accordion>

  <Accordion title="Inflation">
    Inflation is the rate at which the general level of prices for goods and services rises over time, reducing the purchasing power of money. Central banks like the U.S. Federal Reserve monitor inflation closely and adjust interest rates to keep it within a target range.
  </Accordion>

  <Accordion title="Migration">
    Migration is the movement of people from one place to another, whether within a country (internal migration) or across international borders (international migration). It is driven by a range of push factors (like poverty or conflict) and pull factors (like economic opportunity or safety).
  </Accordion>

  <Accordion title="Supply and Demand">
    Supply and demand is a fundamental economic model describing the relationship between the availability of a product (supply) and the desire for it (demand). When demand exceeds supply, prices tend to rise; when supply exceeds demand, prices tend to fall.
  </Accordion>
</AccordionGroup>

## 🏥 Health

<AccordionGroup>
  <Accordion title="Antibody">
    An antibody is a protein produced by the immune system in response to a foreign substance (antigen), such as a virus or bacterium. Antibodies bind to their target antigen to neutralize it or flag it for destruction by other immune cells.
  </Accordion>

  <Accordion title="Antioxidant">
    Antioxidants are compounds that neutralize free radicals — unstable molecules that can damage cells and contribute to aging and disease. Common dietary antioxidants include vitamins C and E, beta-carotene, and polyphenols found in fruits, vegetables, and tea.
  </Accordion>

  <Accordion title="Calorie">
    A calorie is a unit of energy that measures the amount of energy provided by food and beverages. Your body uses calories to fuel every function, from breathing and circulation to exercise and cognitive activity; consuming more calories than you expend leads to weight gain over time.
  </Accordion>

  <Accordion title="Cardio (Cardiovascular Exercise)">
    Cardiovascular exercise — commonly called "cardio" — refers to any sustained physical activity that raises your heart rate and increases oxygen circulation throughout the body. Examples include running, swimming, cycling, and brisk walking; regular cardio improves heart health, endurance, and metabolic function.
  </Accordion>

  <Accordion title="Cholesterol">
    Cholesterol is a waxy, fat-like substance found in your blood and every cell in your body, essential for producing hormones, vitamin D, and bile acids. There are two main types: LDL ("bad" cholesterol), which can build up in arteries, and HDL ("good" cholesterol), which helps remove LDL from the bloodstream.
  </Accordion>

  <Accordion title="Macronutrient">
    Macronutrients are the three main categories of nutrients your body needs in large amounts for energy and growth: carbohydrates, proteins, and fats. Each plays a distinct role — carbohydrates provide quick energy, proteins build and repair tissue, and fats support hormone production and nutrient absorption.
  </Accordion>

  <Accordion title="Metabolism">
    Metabolism refers to the complete set of chemical reactions that occur in living organisms to sustain life, including converting food into energy, building cellular structures, and eliminating waste products. Your basal metabolic rate (BMR) is the number of calories your body burns at rest to maintain basic functions.
  </Accordion>

  <Accordion title="Vaccine">
    A vaccine is a biological preparation that trains the immune system to recognize and fight a specific pathogen — such as a virus or bacterium — without causing the disease itself. Vaccines typically contain weakened or inactivated forms of the pathogen, or fragments of it, that trigger a protective immune response.
  </Accordion>
</AccordionGroup>
